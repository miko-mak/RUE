# Random Urbex Explorer (RUE)
A simple PWA website that lets you explore abandoned places and ruins around the world.

## Installation
This project uses <a href="https://nodejs.org/en/download" target="_blank">Node.js</a>, so make sure you already have it.
Then run **npm install** in the terminal.

## APIs Used
- <a href="https://wiki.openstreetmap.org/wiki/Overpass_API" target="_blank">Overpass API</a> (used for querying and filtering geographic data) 
- <a href="https://nominatim.org/" target="_blank">Nominatim</a> (used to validate city names)

## Tools Used
- <a href="https://leafletjs.com/" target="_blank">LeafLet</a> (used for the world map)
- <a href="https://expressjs.com/" target="_blank">Express.js</a>
